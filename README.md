# TestReactNativeNavigation
ReactNativeNavigation + TypeScript Sample on ReactNative 0.56

# Dependencies

- Mac OS v10.13.4
- XCode v9.4.1
- yarn v1.9.4
- node v8.11.1
- react-native-cli: 2.0.1
- react-native: 0.56.0

# Run App

```
yarn install && react-native run-android
```

or

```
yarn install && react-native run-ios
```
